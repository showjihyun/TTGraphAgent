높은 정확도로 **텍스트 → 지식 그래프 구성 요소(Node, Edge, 속성)**을 추출하려면 아래와 같은 구조적 처리 파이프라인을 따르는 것이 매우 효과적입니다:

✅ 고안한 추출 방법: 정확도 높은 5단계 파이프라인
단계	기능	설명
1️⃣ 전처리	문장 분리 및 정제	문장 단위로 나누고 불필요한 기호 제거
2️⃣ 엔티티 추출 (Node)	Named Entity Recognition (NER)	사람, 조직, 장소, 날짜 등 주요 엔티티 식별
→ Node 생성
3️⃣ 관계 추출 (Edge)	Relation Extraction (RE)	엔티티 쌍 간의 의미 관계 추출
→ Edge 생성
4️⃣ 속성 추출	Attribute Parsing	각 엔티티의 날짜/수치/단위/특성 등 속성 추출
→ 속성 딕셔너리
5️⃣ 구조화 + 검증	LLM 기반 보정	LLM으로 Triple 구조 및 누락 항목 자동 보정
